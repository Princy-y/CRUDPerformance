# Java CRUD Performance Analysis

This project analyzes performance of CRUD operations using Java (JDBC) and MySQL.

## Features
- Insert, Read, Update, Delete operations
- Performance measurement using System.nanoTime()

## Technologies
- Java
- MySQL
- JDBC

## Output
Insert Time: 5399 ms
Read Time: 50 ms
Update Time: 2275 ms
Delete Time: 2729 ms
